## üîç Funci√≥n de Comparaci√≥n de Versiones - Detalles

### Ejemplos de Comparaci√≥n

```javascript
// Comparaciones de versiones principales
compareVersions("2", "1")     // ‚Üí 1  (2.0 > 1.0)
compareVersions("1", "0")     // ‚Üí 1  (1.0 > 0.0)
compareVersions("0", "1")     // ‚Üí -1 (0.0 < 1.0)

// Comparaciones con refinamientos
compareVersions("1.1", "1")   // ‚Üí 1  (1.1 > 1.0)
compareVersions("2.1", "2")   // ‚Üí 1  (2.1 > 2.0)
compareVersions("1", "1.1")   // ‚Üí -1 (1.0 < 1.1)

// Comparaciones entre refinamientos
compareVersions("1.2", "1.1") // ‚Üí 1  (1.2 > 1.1)
compareVersions("1.1", "1.2") // ‚Üí -1 (1.1 < 1.2)

// Comparaciones entre versiones mayores diferentes
compareVersions("2", "1.1")   // ‚Üí 1  (2.0 > 1.1)
compareVersions("2.1", "1.2") // ‚Üí 1  (2.1 > 1.2)
compareVersions("1.1", "2")   // ‚Üí -1 (1.1 < 2.0)

// Comparaciones iguales
compareVersions("1", "1")     // ‚Üí 0  (1.0 = 1.0)
compareVersions("1.1", "1.1") // ‚Üí 0  (1.1 = 1.1)
```

### Algoritmo de Ordenamiento

```javascript
const plans = [
  { planVersion: "0" },
  { planVersion: "1" },
  { planVersion: "1.1" },
  { planVersion: "2" },
  { planVersion: "2.1" }
]

// Ordenar de mayor a menor
const sortedPlans = plans.sort((a, b) =>
  compareVersions(a.planVersion, b.planVersion)
)

// Resultado:
// [
//   { planVersion: "2.1" },  ‚Üê Mayor versi√≥n
//   { planVersion: "2" },
//   { planVersion: "1.1" },
//   { planVersion: "1" },
//   { planVersion: "0" }     ‚Üê Menor versi√≥n
// ]
```
